1—Structure page

Define the size, margins, grid and color of your page. This structure will serve as a base to organize content in a balanced and hierarchical fashion.

The page preview updates as parameters change. You will be able to return to this page at any time to edit these parameters, e.g. increasing the number of columns.

Save and export your project by clicking on the button at the bottom-right corner of the screen. Remember to save and save often!

2—Arrange content

Place and arrange images and text. Manipulate position, scale and typography to compose your message.

Choose to place either text or images. Click on a cell to set a starting position, then, select the size of the element and click again to place it.

Use the edit tool to modify the element's contents and parameters, as well as to change its size and position.

Launch the fullscreen mode to have a closer look at your composition. You can also print the design in this mode.

Save and export your project by clicking on the button at the bottom-right corner of the screen. Remember to save and save often!

3—Define relationships

Set positional constraints between page elements. These relationships describe the visual hierarchy of your page.

Select the first element on the page by double clicking it. Click on a second element to select from a list of possible relationships. These relationships determines the elements' position relative to each other—by default, all elements have a flexible relationship.

These symbols represent vertical relationships. Elements in these relationships are differentiated top-bottom.

These symbols represent horizontal relationships. Elements in these relationships are differentiated left-right.

These symbols represent diagonal relationships. Elements in these relationships are differentiated both top-bottom and left-right.

Save and export your project by clicking on the button at the bottom-right corner of the screen. Remember to save and save often!

4—Explore alternatives

Browse alternatives generated by Whitespace. Explore and edit these variations, while Whitespace keeps track of your design history.

Hit refresh to re-generate alternatives. Pull the slider to set the amount of overlap, or space, between elements. If no alternatives are generated, try reducing the number of relationships, elements or spacing.

Saved alternative will appear in the saved pages drawer at the bottom of the screen. Use this feature to keep a record of your design session.

Saved pages can be edited at any time. You may modify a saved page's structure, contents and relationships to refine its composition.

Save and export your project by clicking on the button at the bottom-right corner of the screen. Remember to save and save often!
